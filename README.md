## Harry Potter Characters App

An Angular application that fetches and displays Harry Potter character data using the [HP API](https://hp-api.onrender.com/).

## Features
- View all Harry Potter characters in a responsive grid
- Click any character to view their full details
- Filter characters by Hogwarts house (Gryffindor, Slytherin, Hufflepuff, Ravenclaw)
- Loading states and error handling
- Built with Angular Material for UI components
- Uses Angular Signals for reactive state management

## Tech Stack
- Angular 21
- Angular Material
- TypeScript
- HP REST API

## How to Run
1. Clone the repository
2. Install dependencies:
 npm install
3. Start the development server:
 ng serve
4. Open your browser at `http://localhost:4200`
